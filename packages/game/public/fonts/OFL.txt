Placeholder for the game's OFL pixel font license.

The app font is wired in packages/game/src/style.css as @font-face 'PixelFont'
with src url('/fonts/pixel.woff2') and a monospace fallback. It is intentionally
absent from the repo for now (a monospace fallback renders meanwhile).

To activate a pixel font:
  1. Choose an OFL-licensed pixel font (e.g. Silkscreen, Press Start 2P).
  2. Drop its .woff2 here as packages/game/public/fonts/pixel.woff2
  3. Replace this file with the font's actual OFL.txt license text.

The Open Font License requires shipping the license alongside the font, which is
why this file lives next to where the .woff2 goes.
